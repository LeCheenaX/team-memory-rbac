# Legacy Documents

This folder keeps historical planning and manual-test drafts only.

Do not use files in this folder as canonical design, configuration, or test
instructions.

Canonical docs now live in exactly three places:

- Design: `docs/design/modules/`
- Configuration: `docs/config/`
- Manual tests: `docs/manual-tests/`

When legacy content is still useful, copy the relevant requirement into one of
the canonical docs and update that canonical doc directly.
